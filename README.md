# sys-opt-tools

## 使用说明
### 安装工具
```bash
# 对应bash环境
bash ./setup.sh

# 对应zsh环境
zsh ./setup.sh
```

### 卸载工具
```bash
# 对应bash环境
bash ./setup.sh --uninstall

# 对应zsh环境
zsh ./setup.sh --uninstall
```
